# ¿Quieres que Viaje?

Este proyecto es una página web interactiva donde puedes hacer una pregunta seria y recibir respuestas de una manera divertida. La página presenta animaciones de confeti y caritas llorando dependiendo de las respuestas dadas.

## Características

- **Interacción con botones**: Los usuarios pueden seleccionar "Sí" o "No" como respuesta.
- **Animaciones**: 
  - Confeti animado cuando se elige "Sí" suficientes veces.
  - Caritas llorando cuando se selecciona "No".
- **Mensajes de Respuesta**: Mensajes aleatorios y específicos dependiendo del número de veces que se haya presionado "Sí".
